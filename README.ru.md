# Alisa Trial Abuser

[English version](README.md)

Приложение открывает страницу регистрации, заполняет нужные данные, выбирает
Windows и показывает полученную VPN-ссылку.

Есть два способа использования:

- GUI-приложение: запустить `alisa-trial-abuser.exe`
- CLI-скрипт: запустить `register_vpn_link.py` из терминала

## Важно

Приложение предоставляется в образовательных целях и должно использоваться
только на сайтах, системах и аккаунтах, где у вас есть разрешение на
автоматизацию. Не используйте его для обхода лимитов, нарушения правил сервисов
или доступа без разрешения.

## Использование GUI

Распакуйте архив приложения и откройте:

```text
alisa-trial-abuser.exe
```

Приложение сначала пытается использовать Microsoft Edge. Если Edge недоступен,
оно пробует Google Chrome. Если ни один браузер не найден, приложение может
скачать Playwright Chromium на этом компьютере.

В окне приложения:

1. Проверьте поля в разделе `Settings`.
2. Включите `Show browser window`, если хотите видеть браузер.
3. Нажмите `Start`.
4. Дождитесь результата.
5. Нажмите `Copy link`, чтобы скопировать ссылку.

## Использование CLI

Сначала установите зависимости:

```powershell
pip install playwright
python -m playwright install chromium
```

Запуск с настройками по умолчанию:

```powershell
python .\register_vpn_link.py
```

Показать окно браузера:

```powershell
python .\register_vpn_link.py --headful
```

Переопределить значения для одного запуска:

```powershell
python .\register_vpn_link.py --password "AnotherPassword123"
```

```powershell
python .\register_vpn_link.py --email-domain "mail.com"
```

Пример вывода:

```text
Generated email: qwerty_ab12cd34@mail.ru
https://sub.example.com/example_link
```

## CLI-Параметры

Все параметры необязательные.

| Параметр | Значение |
| --- | --- |
| `--url` | URL страницы регистрации |
| `--password` | Пароль, который вводится в форму |
| `--email-prefix` | Префикс для сгенерированного email |
| `--email-domain` | Домен для сгенерированного email |
| `--device-text` | Текст для выбора Windows |
| `--browser-channel` | Режим браузера: `auto`, `msedge`, `chrome` или `chromium` |
| `--no-download-chromium` | Отключает fallback со скачиванием Chromium |
| `--headful` | Показывает окно браузера |
| `--timeout-ms` | Максимальное время ожидания действий на странице |
| `--email-selector` | Селектор поля email |
| `--password-selector` | Селектор поля пароля |
| `--submit-selector` | Селектор кнопки отправки формы |
| `--link-selector` | Необязательный селектор поля со ссылкой |
| `--copied-selector` | Необязательный селектор дополнительной кнопки после шага со ссылкой |

## Поля В Окне

`URL` - адрес страницы регистрации.

`Password` - пароль, который вводится в форму.

`Email prefix` - начало сгенерированного email-адреса.

`Email domain` - домен сгенерированного email-адреса.

`Device text` - текст для выбора Windows.

`Browser` - браузер, который использует приложение. `auto` сначала пробует Edge,
затем Chrome.

`Show browser window` - показывает браузер во время работы приложения.

`Download Chromium if needed` - скачивает Playwright Chromium, если Edge и Chrome
недоступны. Для скачивания нужен интернет.

`Timeout ms` - максимальное время ожидания действий на странице.

`Link selector` - можно оставить пустым. Используется только если ссылку нужно
читать из конкретного поля страницы.

## Результат

После успешного запуска приложение показывает или выводит:

`Generated email` - email-адрес, который использовало приложение.

`Link` - VPN-ссылка, найденная приложением.

## Если Что-то Пошло Не Так

Включите `Show browser window` и запустите снова. Так проще понять, где
остановился сценарий.

Если приложение не смогло найти нужный экран или ссылку, оно может создать
скриншот:

```text
no_windows_card_found.png
no_vpn_link_found.png
```

По скриншоту можно понять, что было видно в браузере во время ошибки.
